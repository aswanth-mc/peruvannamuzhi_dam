# Peruvannamuzhi Dam

A comprehensive static website documenting information about Peruvannamuzhi Dam.

## Overview

This repository contains HTML and CSS files for a website dedicated to Peruvannamuzhi Dam. It provides information about the dam's structure, history, and significance.

## Technology Stack

- **HTML** - 66.6% (Markup and structure)
- **CSS** - 33.4% (Styling and layout)
